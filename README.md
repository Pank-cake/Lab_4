# Задание 1. Решить задачу на построение нового дерева по заданному (map)

## Задача 1

### Текст задачи

Дерево содержит вещественные числа. Заменить отрицательные значения на
0, положительные — на 1.

### Алгоритм решения

1) Запрашиваем у пользователя количество элементов дерева.
2) С помощью функции проверяем, чтобы количество было положительным целым числом.
3) С помощью функции рандомно генерируем вещественные значения от -10 до 10:
	- функция при помощи List.init генерирует значения
	- полученные значения с помощью List.fold записываются в дерево через функцию вставки:
																- если узла нет - создаём
																- если число меньше или равно узлу - вставка в левое поддерево 
																- если число больше узла - вставка в правое поддерево
4) Выводим исходное дерево с помощью функции.
5) Переписываем дерево в список с помощью функции.
6) List.map создаёт новый список, заменяя значения списка на 0 при отрицательном числе, на 1 при положительном.
7) Переходим от списка обратно к дереву и выводим с помощью функции.

# Задание 2. Решить задачу на получение заданного результата для указанного дерева (fold)

## Задача 1

### Текст задачи

Сколько элементов дерева содержит заданную цифру?

### Алгоритм решения

1) Запрашиваем у пользователя количество элементов дерева.
2) С помощью функции проверяем, чтобы количество было положительным целым числом.
3) С помощью функции рандомно генерируем вещественные значения от -10 до 10:
	- функция при помощи List.init генерирует значения
	- полученные значения с помощью List.fold записываются в дерево через функцию вставки:
																- если узла нет - создаём
																- если число меньше или равно узлу - вставка в левое поддерево 
																- если число больше узла - вставка в правое поддерево
4) Выводим исходное дерево с помощью функции.
5) Запрашиваем с пользователя цифру и проверяем на корректность с помощью функции.
6) Переписываем цифру в строку.
7) Переписываем дерево в список с помощью функции.
8) List.fold записывает числа из списка в строку и, если находит цифру в числе, то добавляет 1 к аккумулятору.
9) Выводим количество элементов с заданной цифрой.
